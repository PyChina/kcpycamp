# 局域网即时通讯小工具 #
  * 中文全称：局域网即时通讯工具
  * 英文全称：Pythonic Local Area Network Instant Messaging Tool
  * 缩写：PyLanIm
  * 仓库:
    * 开发: [mwkfeng-pylanim - 局域网即时通讯工具 - Google Project Hosting](http://code.google.com/r/mwkfeng-pylanim/source/checkout)
    * 发布: https://bitbucket.org/ZoomQuiet/pycamp.lanim
    * 客户端下载: [.md](.md)https://bitbucket.org/ZoomQuiet/pycamp.lanim/downloads/LanImClient.tbz LanImClient.tbz] <sup>可执行 M$ .exe格式</sup>
  * 动机/目标描述：基于python实现的跨平台简单即时通讯工具
    1. 学习/体验跨平台的网络通讯开发
    1. 实现局域网内客户端之间的通信
    1. 包括文字通讯和文件传输
  * 部署/运行环境约束：跨平台的局域网应用程序
    * ~ ZoomQuiet : 建议除Py 环境之外不依赖其它任何系统
